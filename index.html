Ai-Workout-Gen:

Workout Generator is a dynamic full-stack application designed to create personalized workout plans. It utilizes React for the front-end and Node.js with Express for the server-side. It leverages the ChatGPT API to generate custom workout routines based on user responses to targeted questions about their fitness preferences and habits. It prompts the ChatGPT API to return the response in a json format to get parsed. If in case the response is not in the desired format, I have made my own algorithm to parse the data using regex so that the result can be easily displayed to the user. The application ensures user security with a robust signup-login system that encrypts passwords using MySQL, providing both functionality and protection.

AI-NBA Game Predictor:

The AI-NBA Bot Predictor leverages Python, TensorFlow, and Keras to forecast NBA game outcomes with a 67% accuracy for the 2023-2024 season. It processes about 1,200 games, using detailed data scraped from NBA reference sites and parsed with BeautifulSoup. The model efficiently maps players to unique IDs, creates embeddings, and builds a neural network that considers team dynamics and starters. An early stopping mechanism in training ensures optimal prediction without overfitting
